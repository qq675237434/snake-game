## 🐛 常见问题排查

### 问题 1: 按钮不可见
**检查：**
1. 浏览器控制台是否有 JavaScript 错误
2. CSS 是否正确加载（检查网络面板）
3. `overlay-buttons` 类的 display 属性

**修复：**
```javascript
// 在浏览器控制台执行
document.querySelector('.overlay-buttons').style.display = 'flex';
```

### 问题 2: 按钮被遮挡
**检查：**
1. z-index 层级是否正确
2. 内容区域是否设置了 overflow

**修复：**
```css
.overlay-buttons {
    z-index: 100; /* 确保在最上层 */
}
```

### 问题 3: 移动端按钮太小
**检查：**
1. 媒体查询是否正确应用
2. 按钮最小尺寸

**修复：**
```css
@media screen and (max-width: 480px) {
    .overlay-buttons .btn {
        min-width: 120px;
        min-height: 44px; /* 触摸友好尺寸 */
    }
}
```

## 🔧 调试技巧

### 查看按钮层级
```javascript
// 浏览器控制台
console.log(document.querySelector('.overlay-buttons'));
console.log(getComputedStyle(document.querySelector('.overlay-buttons')).display);
```

### 模拟不同状态
```javascript
// 强制显示按钮栏
document.querySelector('.overlay-buttons').style.display = 'flex';

// 强制隐藏按钮栏
document.querySelector('.overlay-buttons').style.display = 'none';

// 模拟游戏结束
window.game.gameOver();
```

### 检查事件绑定
```javascript
// 查看按钮事件监听器
getEventListeners(document.getElementById('startBtn'));
```
